Chapter 5

Notes From Chapter 5:

  • JavaScript objects are a set of self-contained values and functions. A function that is the property of an object is called its method.

  • An object literal is an object that is created by wrapping all its properties and methods in curly braces. Each property is a key-value pair, separated by commas.

  • To create an object literal, enter a pair of curly braces after the variable name. const game = {};

  • The chapter spent a lot of time teaching regular expressions const pattern = /[a-zA-Z]+ing$/;

Chapter 6

Notes From Chapter 6:

  • Getting elements:

    • getElementById() "Returns a reference to the element with a unique id attribute that is given as an argument"
    • getElementByClassName() "will return a live node list of all elements that have the class name that is supplied as an argument"
    • getElementsByTagName() “will return a live node list of all the elements with the tag name that is provided as an argument.”
    • document.querySelector() “method allows you to use CSS notation to find the first element in the document that matches a CSS selector provided as an argument. “
    • document.querySelectorAll() “returns a node list of all the elements in the document that match the CSS query selector.”

  • Getting and Setting Attributes

    • getAttribute() “returns the value of the attribute provided as an argument.”
    • setAttribute() “changes the value of an element’s attributes. It takes two arguments: the attribute that you wish to change and the new value of that attribute.”

  • Classes Of An Element

    • className “allows the class of an element to be set directly. Also, it can be used to find out the value of the class attribute:” Be careful with this one; it will override all the classes to add the new one.
    • classList.add() is a better way to add a class because it won’t override existing classes. classList.toggle() will toggle the class on and off
    • Creating an element createElement()

  • Creating a Text Node

    • createTextNode() The text goes inside the parenthesis with quotation marks
    • when you use createTextNode it doesn't link to it's parent automaticly textContent does and it's simpler to use. here is an example of how to create an element and fill it with text: const flash = document.createElement('li'); flash.textContent = 'Flash';
    • This is an example of how to create and fill an element with a function: function createElement (tag,text) { const el = document.createElement(tag); el.textContent = text; return el }
